# Express.js Basics

This project provides a basic introduction to Express.js, a minimal and flexible Node.js web application framework.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/Awais9986/Express-js-Basics.git
   ```

2. Navigate to the project directory:
   ```
   cd express-js-basics
   ```

3. Install dependencies:
   ```
   npm install
   ```

## Usage

1. Start the server:
   ```
   npm start
   ```

2. Open your browser and visit `http://localhost:3000` to see the application in action.

## Features

- Basic routing
- Middleware usage
- Serving static files
- Error handling
